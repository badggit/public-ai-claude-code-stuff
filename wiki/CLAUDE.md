## Session Initialization

At the start of each session, you MUST:
1. **Read `docs/wiki/README.md`** to review the current documentation index
2. **Memorize the available documentation files** and their topics
3. **Use existing documentation** when encountering questions covered in the docs index
4. **Reference specific documentation files** (e.g., `docs/wiki/feature-wipe-and-reload.md`) when relevant topics arise

This ensures you leverage existing project knowledge instead of rediscovering information.
