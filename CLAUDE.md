# AI & Claude Code utilities

This project contains utilities for improving work with AI and Claude Code.

## Contents

### /wiki
A documentation management system for Claude Code that allows creating and maintaining project documentation with an automated update command.

To use the wiki system:
1. Add the contents of `wiki/CLAUDE.md` to your project's CLAUDE.md file
2. Copy `wiki/.claude` and `wiki/docs` to your project's root directory - `/.claude` and `/docs`
3. Use the `/wiki-update` command in Claude Code to update documentation

## Purpose

This repository serves as a collection of practical tools to enhance productivity when working with AI assistants and Claude Code specifically.
